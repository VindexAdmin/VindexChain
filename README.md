# VindexChain 🔗🔥

**VindexChain** es una blockchain construida desde cero en Go, desarrollada como una respuesta a la injusticia financiera de los bancos tradicionales.  
Este proyecto busca devolver el poder económico al pueblo de Puerto Rico mediante tecnología descentralizada y transparente.

## Características actuales
- Bloques enlazados con hash
- Transacciones básicas entre direcciones
- Bloque génesis funcional
- Estructura modular en Go

## Próximas mejoras
- Wallet con clave privada/pública
- Firma de transacciones
- Red P2P para nodos
- API REST para interacción externa
- Explorador de bloques (**VindexScan**)

## Filosofía
Este proyecto nace desde la necesidad de justicia financiera y representa una revolución contra el sistema bancario injusto.  
Es tiempo de empoderar al pueblo con una herramienta que no puede ser controlada por unos pocos.
